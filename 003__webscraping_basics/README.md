# Bases du webscraping

## Instruction

### Cloner le répertoire Git

```bash
git clone https://github.com/agailloty/ALGOPY-26.git
```

### Se placer dans le répertoire du projet

```bash
cd ALGOPY-26\003__webscraping_basics
```

### Installer le packages

```bash
uv sync
```

### Démarrer Jupyter Lab
Assurez-vous d'être dans le dossier `003__webscraping_basics`
```bash
uv run jupyter lab
```
